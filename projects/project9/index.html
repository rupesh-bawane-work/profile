Social Network Analysis

Community Detection • Influence Analysis • Network Visualization

Business Problem

Large-scale social networks contain hidden structures such as communities, influential users, and bridge connectors. Traditional analysis fails to uncover influence concentration, information flow paths, and segmentation opportunities, limiting data-driven decision making.

Project Overview

This project applies graph-based network analysis using Python and NetworkX to identify community structures, influential nodes, and bridge connectors in a social network. The analysis supports influence maximization, segmentation, and network resilience assessment.

Data Requirements

  • Edge list files (node-to-node relationships)
  • Community circle definitions
  • Node-level feature files
  • Ego-network attributes

Graph Construction

Python 
import pandas as pd
import networkx as nx

# Load edge list
edges = pd.read_csv(
    "facebook_combined.txt",
    sep=" ",
    header=None,
    names=["node_1", "node_2"]
)

# Create graph
G = nx.Graph()
G.add_edges_from(edges.values)

G.number_of_nodes(), G.number_of_edges()

Bridge Node Detection

Bridge nodes connect multiple communities and play a critical role in information diffusion and influence spread.

Python 
bridge_nodes = []

for node in G.nodes():
    neighbor_communities = {
        G.nodes[n].get("community", -1)
        for n in G.neighbors(node)
    }
    if len(neighbor_communities) > 1:
        bridge_nodes.append(node)

len(bridge_nodes)

communities = list(

Network Visualization

Python 
import matplotlib.pyplot as plt

pos = nx.spring_layout(G, seed=42)
node_colors = [G.nodes[n].get("community", 0) for n in G.nodes()]

plt.figure(figsize=(12, 10))
nx.draw(
    G,
    pos,
    node_color=node_colors,
    cmap=plt.cm.tab20,
    node_size=200,
    edge_color="gray",
    alpha=0.6
)
plt.title("Network Communities & Structure")
plt.axis("off")
plt.show()
